IMO 2016 Problem 2

Fill N\u00d7N Table with I,M,O Equally

Problem Statement

Find all integers n for which each cell of an n × n table can be filled with one of the letters A, B, and C in such a way that:

  • In each row and each column, one third of the entries are A, one third are B, and one third are C; and
  • In any diagonal, if the number of entries on the diagonal is a multiple of three, then one third of the entries are A, one third are B, and one third are C.

Note: The rows and columns of an n × n table are each labelled 1 to n in a natural order. For n ≥ 2, the table has 4n-2 diagonals of two types:

  • Type 1: All cells (i,j) for which i-j is constant
  • Type 2: All cells (i,j) for which i+j is constant

Theoretical Result

According to the IMO solution, n is valid if and only if n ≡ 0 (mod 3) and n ≢ 0 (mod 9).

This means valid values are: n = 3, 6, 12, 15, 21, 24, 30, ...

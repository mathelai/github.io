This is a solution to the IMO problem.

Let $M=2015$. We are given a sequence of integers $a_1, a_2, \dots$ satisfying:
(i) $1 \le a_j \le M$ for all $j \ge 1$.
(ii) $k+a_k \ne \ell+a_\ell$ for all $1 \le k < \ell$.

We want to prove the existence of positive integers $b$ and $N$ such that
$$
\left|\sum_{j=m+1}^{n}\left(a_{j}-b\right)\right| \le 1007^{2}
$$
for all integers $n>m \ge N$. Note that $1007 = (M-1)/2$.

Let $x_k = k+a_k$. Condition (ii) states that the sequence $(x_k)_{k\ge 1}$ consists of distinct positive integers. Condition (i) implies $k+1 \le x_k \le k+M$.

**Step 1: Identifying the constant b.**

Let $X = \{x_1, x_2, \dots\}$. Let $Y = \mathbb{Z}^+ \setminus X$ be the set of "holes".
We first show that $Y$ is finite. For any large integer $L$, consider the interval $[1, L]$. If $k \le L-M$, then $x_k \le k+M \le L$. Thus, $\{x_1, \dots, x_{L-M}\} \subseteq [1, L]$.
The number of elements in $X$ up to $L$ is $|X \cap [1, L]| \ge L-M$.
The number of holes up to $L$ is $|Y \cap [1, L]| = L - |X \cap [1, L]| \le M$.
Since this bound is independent of $L$, the set $Y$ is finite.

Let $H = |Y|$. Since $x_k \ge 2$, $1 \in Y$. So $H \ge 1$. We choose $b=H$.

**Step 2: Analyzing the partial sums using relative holes.**

Let $X_n = \{x_1, \dots, x_n\}$. We have $X_n \subset \{1, \dots, n+M\}$.
Define the set of relative holes $Y_n = \{1, \dots, n+M\} \setminus X_n$. The size of $Y_n$ is constant: $|Y_n| = M$.

Let $E_n = \sum_{j=1}^n a_j$. Let $S(A)$ denote the sum of elements in a finite set $A$. We relate $E_n$ to $S(Y_n)$.
$$
\sum_{j=1}^{n+M} j = S(X_n) + S(Y_n) = \sum_{j=1}^n x_j + S(Y_n) = \sum_{j=1}^n (j+a_j) + S(Y_n).
$$
$$
\frac{(n+M)(n+M+1)}{2} = \frac{n(n+1)}{2} + E_n + S(Y_n).
$$
$$
E_n = \frac{(n+M)(n+M+1) - n(n+1)}{2} - S(Y_n).
$$
$$
E_n = \frac{2nM+M^2+M}{2} - S(Y_n) = nM + \frac{M(M+1)}{2} - S(Y_n).
$$

We analyze the partial sum from $m+1$ to $n$:
$$
\sum_{j=m+1}^n a_j = E_n-E_m = (n-m)M + S(Y_m) - S(Y_n).
$$
We want to bound the quantity:
$$
Q(m,n) = \left|\sum_{j=m+1}^{n}\left(a_{j}-H\right)\right| = |E_n-E_m - (n-m)H|.
$$
$$
Q(m,n) = |(n-m)(M-H) + S(Y_m) - S(Y_n)|.
$$
Let $K=M-H$. Since $1 \le H \le M$, we have $0 \le K \le M-1$.

**Step 3: Analyzing the configuration of holes for large n.**

Since $Y$ is finite, let $N = \max(Y)$.
If $n \ge N$, since $M\ge 1$, $n+M > N$. Thus $Y \subset \{1, \dots, n+M\}$. Since $Y \cap X_n = \emptyset$, we have $Y \subset Y_n$.

We partition $Y_n$ as $Y_n = Y \cup Z_n$, where $Z_n = Y_n \setminus Y$ are the transient holes.
$|Z_n| = |Y_n|-|Y| = M-H = K$.

We analyze the location of $Z_n$. $Z_n \subset X$ and $Z_n \cap X_n = \emptyset$.
So $Z_n \subset \{x_{n+1}, x_{n+2}, \dots\}$.
If $z \in Z_n$, $z=x_k$ for $k \ge n+1$. Thus $z \ge k+1 \ge n+2$.
Since $Z_n \subset Y_n$, $Z_n \subset \{1, \dots, n+M\}$.
Therefore, $Z_n \subset \{n+2, \dots, n+M\}$.

**Step 4: Bounding the partial sums.**

For $n>m \ge N$, we look at $Q(m,n)$.
$S(Y_n) = S(Y) + S(Z_n)$.
$$
Q(m,n) = |(n-m)K + S(Z_m) - S(Z_n)|.
$$
$Z_n$ is a subset of $\{n+2, \dots, n+M\}$ of size $K$.
We can write $S(Z_n) = nK + C(Z_n)$, where $C(Z_n)$ is the sum of the offsets of the elements in $Z_n$ relative to $n$. $C(Z_n)$ is the sum of $K$ distinct integers in $\{2, \dots, M\}$.

Substituting this back into $Q(m,n)$:
$$
Q(m,n) = |(n-m)K + (mK+C(Z_m)) - (nK+C(Z_n))| = |C(Z_m) - C(Z_n)|.
$$
To bound this difference, we find the range of possible values for $C(Z_n)$.
Let $C_{min}(K)$ be the minimum sum, achieved by the set $\{2, \dots, K+1\}$.
$$
C_{min}(K) = \sum_{j=2}^{K+1} j = \frac{K^2+3K}{2}.
$$
Let $C_{max}(K)$ be the maximum sum, achieved by the set $\{M-K+1, \dots, M\}$.
$$
C_{max}(K) = \sum_{j=M-K+1}^{M} j = KM - \frac{K(K-1)}{2}.
$$
The maximum difference is $B(K) = C_{max}(K) - C_{min}(K)$.
$$
B(K) = \left(KM - \frac{K^2-K}{2}\right) - \frac{K^2+3K}{2} = KM - \frac{2K^2+2K}{2}.
$$
$$
B(K) = KM - K^2 - K = K(M-K-1).
$$
We have $Q(m,n) \le B(K)$.

**Step 5: Maximizing the bound.**

We maximize $B(K)$ over $0 \le K \le M-1$. This quadratic function is maximized when $K$ is closest to $(M-1)/2$.
With $M=2015$, $(M-1)/2 = 2014/2 = 1007$.
The maximum value is achieved at $K=1007$.
$$
B(1007) = 1007(2015-1007-1) = 1007(1007) = 1007^2.
$$
Thus, we have proved that there exist positive integers $b=H$ and $N$ such that for all $n>m \ge N$,
$$
\left|\sum_{j=m+1}^{n}\left(a_{j}-b\right)\right| \le 1007^{2}.
$$
